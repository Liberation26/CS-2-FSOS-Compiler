# Oryn 0.3.4 Release Notes

## Stage 3 feature parity proof

This release expands the Stage 3 kernel and test suite so the direct ELF64 object writer proves parity with the useful Stage 2 compiler subset.

Stage 3 now deliberately exercises:

- local integer declarations,
- local stores and loads,
- integer constants,
- string constants,
- addition,
- subtraction,
- equality comparison,
- less-than comparison,
- `while` loops,
- `if` / `else` branches,
- approved module calls,
- static helper method calls,
- explicit `return`,
- final CPU halt.

## Runtime proof lines

The Stage 3 QEMU proof now checks the added parity diagnostics:

```text
Stage3 subtraction worked
Stage3 integer arithmetic worked
Stage3 explicit return worked
Stage3 parity proof complete
```

## New test

Added:

```text
Tests/Compiler/Stage3/09-stage3-feature-parity-check.sh
```

The new test inspects the generated Stage 3 IR and readable assembly reference to prove the expected opcodes, helper methods, approved calls, and string literals are present before the QEMU boot proof runs.

## Updated tests

Updated:

```text
Tests/Compiler/Stage3/03-qemu-stage3-boot.sh
Tests/Compiler/Stage3/06-elf64-symbol-check.sh
Tests/Compiler/Stage3/07-elf64-relocation-check.sh
Tests/Compiler/Stage3/run.sh
```

The symbol and relocation checks now include the additional `Kernel_WriteReturnProof` helper method emitted by the parity kernel.
