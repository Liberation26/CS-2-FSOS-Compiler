# Oryn 0.2.11

## Stage 2 Phase 6 - Stack/local variable model

- Added 64-bit rbp-relative stack slots for Stage 2 integer locals.
- Updated generated x64 assembly to use a conventional prologue and `leave`/`ret` epilogue.
- Kept Stage 2 local slots simple: every integer local uses one 64-bit slot, even when the C# source type is `int`.
- Added Stage 2 test checks for stack-frame setup, local slot offsets, local zero-initialisation, and epilogue generation.
- Bumped Oryn and Runqemu versions to 0.2.11.
