# Oryn 1.0.4 Release Notes

Oryn 1.0.4 is a corrective build for the first end-user OS generation milestone.

## Fixed

- Fixed generated JSON compatibility with Python readers by writing generator output as UTF-8 without a byte order mark.
- Updated `Oryn.sh` and `Runqemu.sh` JSON readers to accept existing UTF-8 BOM files using `utf-8-sig`.
- Updated generator tests so manifest and answer JSON parsing is BOM-tolerant.
- Bumped visible version strings to 1.0.4.

## Policy retained

- Diagnostics and Panic are mandatory and always enabled.
- User-selected modules exclude mandatory modules required to get the kernel running.
