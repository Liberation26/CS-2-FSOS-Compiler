# Oryn 0.1.3 Release Notes

Oryn 0.1.3 fixes the QEMU launch behaviour after the first Stage 1 backend run.

## Changed

- Bumped root `VERSION` to `0.1.3`.
- Bumped `update.sh` version to `0.1.3`.
- Bumped `Runqemu.sh` version to `0.1.3`.
- Bumped `Oryn.Compiler` version output to `0.1.3`.
- `Runqemu.sh` now launches QEMU in headed mode by default.
- Added `ORYN_QEMU_DISPLAY=headless` and `ORYN_QEMU_HEADLESS=1` support for non-graphical runs.
- Kept the bounded timeout behaviour because the proof kernel intentionally halts forever.

## Expected proof

After `update.sh` completes, it calls `Runqemu.sh`. `Runqemu.sh` should build the Stage 1 freestanding kernel and launch QEMU with a visible window by default.
