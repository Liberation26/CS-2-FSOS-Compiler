# Oryn 0.2.17

Stage 2 test and documentation package.

## Added

- Added `Tests/Compiler/Stage2/01-compile-stage2-kernel.sh`.
- Added `Tests/Compiler/Stage2/02-ir-output-check.sh`.
- Added `Tests/Compiler/Stage2/03-assembly-output-check.sh`.
- Added `Tests/Compiler/Stage2/04-qemu-stage2-boot.sh`.
- Updated `Tests/Compiler/Stage2/run.sh` to run the full Stage 2 test set.
- Added Stage 2 user-facing manual coverage under `Manual/README.md`.

## Tests now check

- compiler/run pipeline exits with status 0,
- IR file exists,
- assembly file exists,
- ELF exists,
- ISO exists,
- QEMU reaches the expected diagnostics,
- timeout after the halt loop is treated as success.

## Documentation

The documentation now clearly states:

- Stage 1 proves approved calls can become a bootable freestanding kernel.
- Stage 2 proves Oryn can compile a useful C# subset with variables, branches, loops, helper methods, and module calls.
