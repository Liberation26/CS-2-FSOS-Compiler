# Oryn 0.6.0 Release Notes

## Stage 6 complete

Oryn 0.6.0 adds Stage 6: service/module manifest loading.

## Added

- JSON module manifests under `Source/Sdk/ModuleManifests/`.
- `ModuleManifestCatalog` in the compiler.
- Manifest-backed compiler diagnostics for exposed safe kernel modules.
- `ManifestLoader.InitializeSelected()` SDK and binding.
- Generated native manifest glue for Stage 6 builds.
- Stage 6 kernel and Stage 6 test scripts.

## Proven

- Manifest metadata exposes approved modules to the compiler.
- Manifest metadata drives native module linking.
- Manifest metadata drives initialization order through generated freestanding C glue.
- The Stage 6 kernel boots and reaches expected manifest diagnostics before halting forever.
