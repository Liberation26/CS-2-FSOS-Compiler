# Oryn 1.0.5 release notes

Oryn 1.0.5 fixes generated-OS manifest source path handling after the 1.0.4 BOM compatibility change.

## Fixes

- `Runqemu.sh` now writes `${STAGE_NAME}.manifest.sources` as UTF-8 without BOM.
- Native module source paths are now trimmed for a leading UTF-8 BOM before existence checks and compilation.
- Manifest metadata `nativeSource` values are stripped of accidental leading BOM characters before source path expansion.
- Bumped `Oryn.Compiler` runtime version reporting to 1.0.5 so compiler diagnostics match the delivered archive version.
