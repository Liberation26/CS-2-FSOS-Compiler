# Oryn 0.2.7

## Stage 2 Phase 4: Control Flow Graph

This release adds the first real Stage 2 control-flow graph.

Changes:

- Added basic-block splitting in `ControlFlowGraphBuilder`.
- Added successor-edge generation for `Jump`, `JumpIfFalse`, and fallthrough paths.
- Changed generated labels to readable names such as `LoopStart0`, `LoopEnd0`, `IfElse1`, and `IfEnd1`.
- Added the full `ControlFlowGraph` to the Stage 2 IR manifest.
- Added `[ CFG ]` diagnostics output for generated blocks and successors.
- Updated the Stage 2 sample kernel to exercise `while (Counter < 3)` and `if` after the loop.

This is deliberately not an optimiser. It is a reliability and readability step so later native backend work can consume explicit branch structure.
