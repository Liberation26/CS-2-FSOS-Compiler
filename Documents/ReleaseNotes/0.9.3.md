# Oryn 0.9.3 Release Notes

Oryn 0.9.3 fixes a Stage 9 compiler build failure in the module API contract catalog.

## Fixed

- Fixed `ModuleApiContractCatalog.cs` so the `argumentTypes` fallback uses a compatible collection type.
- This resolves the C# compile error:

```text
Operator '??' cannot be applied to operands of type 'List<string>' and 'string[]'
```

## Stage 9 intent retained

Stage 9 still proves:

- selected modules are composed into a generated kernel source from a template;
- approved module API calls are allowed;
- invalid/unapproved calls are rejected by the compiler before backend/native compilation;
- invalid-call failure must not produce generated native artifacts.
