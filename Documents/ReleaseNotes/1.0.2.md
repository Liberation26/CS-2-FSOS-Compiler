# Oryn 1.0.5 Release Notes

Oryn 1.0.5 is a corrective build for the first end-user OS generation milestone.

## Fixed

- Fixed the `Oryn.Generator` compile failure caused by an interpolated raw string literal containing generated C# braces.
- Replaced the affected generated kernel stub builder with explicit line-based string construction so the generated `Kernel.cs` source can contain literal braces safely.
- Updated generator, script, and documentation version text to 1.0.5.
- Clarified that GitHub push authentication failures leave the local commit intact and are not Oryn build failures.

## Policy retained

- Mandatory kernel modules remain hidden from the user-selected module list.
- Diagnostics and Panic remain always enabled.
- User-selected modules still exclude the modules required to get the kernel running.
