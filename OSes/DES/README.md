# DES

Generated by Oryn 1.0.8.

Kernel name: `DESKernel`

## Build and run

From the Oryn repository root:

```bash
./Oryn.sh build DES
./Oryn.sh run DES
```

The generated manifest records mandatory kernel modules separately from user-selected modules. Runtime, Diagnostics, Panic, Cpu, and ManifestLoader are linked automatically. Diagnostics and Panic are always enabled. The generated kernel prints `Hello from DES` during boot. The VM display mode is chosen during generation and can be Headless or Visual. Headless mode closes automatically after the proof timeout. Visual mode stays open until you close the VM window.
