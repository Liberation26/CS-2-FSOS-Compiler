# Oryn 1.0.4 Release Notes

Oryn 1.0.4 is a corrective build for the first end-user OS generation milestone.

## Fixes

- Fixed `Oryn.Generator/Program.cs` project-root detection.
- Removed the local `Directory` variable name that shadowed `System.IO.Directory`.
- Corrected `Directory.Exists(...)` usage so it resolves to `System.IO.Directory.Exists(...)`, not `DirectoryInfo.Exists`.
- Updated visible version strings to 1.0.4.

## Notes

GitHub push failures that mention invalid username or token are authentication/token issues. The local commit still exists and Oryn generation can still proceed after the build is fixed.
