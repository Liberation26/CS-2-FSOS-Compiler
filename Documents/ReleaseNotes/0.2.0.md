# Oryn 0.2.0

## Stage 2 part 1

- Added `OSes/Stage2/Source/Kernel.cs` as the first Stage 2 kernel source.
- Added `OSes/Stage2/README.md` explaining the Stage 2 purpose, current proof, and next compiler steps.
- Added `Tests/Compiler/Stage2/README.md` as the Stage 2 test area baseline.
- Updated `Runqemu.sh` to accept `Stage1` or `Stage2` as a stage selector.
- Changed the default `Runqemu.sh` target to `Stage2`, while keeping Stage 1 runnable as a regression proof.
- Bumped compiler, updater, and repository version metadata to `0.2.0`.

## Note about constants

`ConstInt32 0` is an IR type marker for the C# `int` literal. It does not force the x64 backend to use a wasteful encoding when a smaller machine immediate is valid.
