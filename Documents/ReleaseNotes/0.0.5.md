# Oryn 0.0.5 Release Notes

## Update workflow

- Updated the default GitHub origin remote to `https://github.com/Liberation26/CS-2-FSOS-Compiler.git`.
- `update.sh` displays version `0.0.5` at startup.
- `update.sh` continues to select the highest semantic `Oryn-x.y.z.zip` archive from `~/Downloads` when no archive path is supplied.
- `update.sh` continues to extract the selected archive to `/tmp`, self-reset when the archive contains a different updater, copy `ChangedFiles/` into `~/Dev/OrynFoundry`, commit, and attempt to push.
