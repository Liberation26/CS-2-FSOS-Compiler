# Oryn 1.0.0 release notes

Oryn 1.0.0 is the first end-user OS generation milestone.

## Added

- Added `Oryn.sh` user workflow wrapper.
- Added `Source/Core/Oryn.Generator`.
- Added JSON-backed question files under `Questions/`.
- Added generated OS folder creation under `OSes/<OsName>/`.
- Added generated OS manifest files separating mandatory kernel modules from user-selected modules.
- Added user OS build/run support through `Runqemu.sh <OsName>`.
- Added 1.0.0 generator tests under `Tests/Generator/1.0.0/`.
- Added documentation for generating, building, and running the first generated OS.

## Changed

- Bumped Oryn.Compiler and package version to 1.0.0.
- Kernel composition now distinguishes mandatory kernel modules from user-selected modules.
- Diagnostics and Panic are always enabled and must not be listed as user-selected modules.

## Module policy

Mandatory kernel modules:

- Runtime
- Diagnostics
- Panic
- Cpu
- ManifestLoader

User-selectable modules for 1.0.0:

- Memory
