# Oryn 0.1.4 Release Notes

## Summary

This release makes the Stage 1 C# kernel source visible as a generated OS tree under `OSes/Stage1`.

## Changes

- Added `OSes/Stage1/Source/Kernel.cs` as the default Stage 1 kernel source file.
- Added `OSes/Stage1/README.md` to explain the Stage 1 source and build-output locations.
- Updated `Runqemu.sh` to compile `OSes/Stage1/Source/Kernel.cs` by default.
- Updated `Runqemu.sh` to write its build output under `OSes/Stage1/Build/Runqemu` by default.
- Bumped root, compiler, updater, and runner versions to `0.1.4`.

## Notes

The old compiler test input under `Source/Core/Oryn.Compiler/Tests/Stage0/` remains available as a compiler test fixture, but it is no longer the default kernel used by `Runqemu.sh`.
