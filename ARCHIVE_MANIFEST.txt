Oryn 1.0.7

Package layout:
- FullSource/ contains the complete Oryn source tree for this delivery.
- ChangedFiles/ contains only files changed for the 1.0.7 delivery.

1.0.7 focus:
- Generated kernels print `Hello from <OsName>` during boot.
- The generator asks whether the VM display should be Headless or Visual.
- Generated manifests and answer files include VmDisplayMode.
- Oryn.sh uses the generated manifest display mode when running the OS.
- Runqemu.sh accepts visual/headed/gui aliases and validates the generated OS hello proof line.
