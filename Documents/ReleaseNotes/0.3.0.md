# Oryn 0.3.0

Stage 3 introduces the direct ELF64 relocatable object writer.

## Added

- `NativeElf64ObjectEmitter`, replacing the previous text placeholder object output.
- Direct `.o` generation with `.text`, `.rodata`, `.rela.text`, `.symtab`, `.strtab`, `.shstrtab`, and `.note.GNU-stack`.
- Stage 3 OS proof under `OSes/Stage3/`.
- Stage 3 compiler tests under `Tests/Compiler/Stage3/`.

## Changed

- `Runqemu.sh` now defaults to Stage 3.
- Stage 3 links the direct Oryn-generated ELF64 object instead of assembling the generated `.S` reference file for the kernel object.
- Compiler diagnostics now identify the direct ELF64 object writer path.

## Proof target

Stage 1 proves approved calls can become a bootable freestanding kernel.

Stage 2 proves Oryn can compile a useful C# subset with variables, branches, loops, helper methods, and module calls.

Stage 3 proves Oryn can directly write a real ELF64 relocatable object from Oryn IR and link it into a bootable freestanding x86_64 kernel.
