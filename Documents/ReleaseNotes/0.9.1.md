# Oryn 0.9.1 Release Notes

## Stage 9 Runqemu dispatcher fix

Oryn 0.9.1 fixes the root `Runqemu.sh` stage dispatcher so `./Runqemu.sh Stage9` and `./Runqemu.sh 9` are accepted.

## Fixed

- Added the missing `Stage9` case to the root Runqemu stage selector.
- Updated unsupported-stage guidance to include `Stage9`.
- Updated Stage1 refusal guidance to describe Stage 9 development mode.
- Bumped script/package metadata to 0.9.1.

## Expected proof

The previous failure:

```text
[FAIL] [ RUNQEMU  ] Unsupported stage: Stage9. Use All, Stage2, Stage3, Stage4, Stage5, Stage6, Stage7, or Stage8.
```

should now proceed into:

```text
[ OK ] [ RUNQEMU  ] Selected stage: Stage9
[ OK ] [ RUNQEMU  ] Generating Stage9 kernel source from selected module template
```
