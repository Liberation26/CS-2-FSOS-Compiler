# Oryn 0.4.0 Release Notes

## Added

- Stage 4 approved module boundary.
- Approved module catalogue metadata in `Source/Sdk/Bindings/*.binding.json` for namespace, type, method, signature, argument types, native symbol, stage, and approval status.
- Compiler validation for approved module namespaces and calls.
- Clear Stage 4 diagnostics for unapproved calls and unapproved `Oryn.Kernel.*` namespaces.
- Stage 4 example kernel under `OSes/Stage4/`.
- Stage 4 tests proving approved calls compile and forbidden calls fail.

## Updated

- Compiler version bumped to `0.4.0`.
- `Runqemu.sh` defaults to Stage 4 and accepts `Stage4`/`4`.
- README rewritten to explain the current Stage 3 capability and the new Stage 4 boundary.
