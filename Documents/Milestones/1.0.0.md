# Oryn 1.0.3 milestone

Oryn 1.0.3 is the first end-user OS generation milestone.

It turns the Stage 9 generated-kernel proof into a user-facing workflow:

```text
JSON questions
  -> answer file
  -> OSes/<OsName>/
  -> kernel template
  -> generated kernel source
  -> safe-subset validation
  -> approved-call/API-contract validation
  -> native freestanding object
  -> linked bootable ISO
  -> QEMU proof
```

## Module policy

Mandatory modules are linked automatically and hidden from user selection:

- Runtime
- Diagnostics
- Panic
- Cpu
- ManifestLoader

Diagnostics and Panic are always enabled.

User-selected modules are optional generated-OS modules. For 1.0.3, Memory is the only available user-selectable module.
