# Oryn 1.0.7

Oryn 1.0.7 adds the generated OS greeting and VM display-mode question.

## Changes

- Generated kernels now print `Hello from <OsName>` during boot.
- The end-user generator now asks whether the VM should be `Headless` or `Visual`.
- Generated answer and manifest JSON now include `VmDisplayMode`.
- `Oryn.sh run <OsName>` now drives QEMU display mode from the manifest unless explicitly overridden.
- `Runqemu.sh` accepts `visual`, `visible`, `headed`, `head`, and `gui` as visual display aliases.
- Generated OS proof checks now require the hello line for user-generated OS runs.

## Expected generated OS proof line

```text
[SERIAL] [ OK ] [ KERNEL   ] Hello from <OsName>
```
