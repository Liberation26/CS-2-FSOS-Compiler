# Oryn 2.0.2 Release Notes

Oryn 2.0.2 fixes the first browser-based OrynVisualConfigurator compile issue.

## Fixed

- Fixed the `RenderPage` call for the unknown-route page so it passes the required navigation argument separately.
- Kept `OrynVisualConfigurator` reading the existing `Questions/*.question.json` files instead of hardcoding questions.
- Kept visual configuration as a local browser UI with terminal fallback.

## Verification

- Shell syntax checks for `Oryn.sh`, `Runqemu.sh`, and `update.sh`.
- Static configurator compile-shape regression checks under `Tests/Generator/2.0.2/`.
