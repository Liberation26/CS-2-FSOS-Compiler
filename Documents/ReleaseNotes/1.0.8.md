# Oryn 1.0.8

Oryn 1.0.8 improves the first end-user OS generation flow.

## Changes

- Visual VM mode no longer uses the headless proof timeout. QEMU stays open until the user closes the VM window.
- Headless VM mode still uses the proof timeout and treats a running halted kernel as success.
- Generator questions now show the expected answer format and default value before asking for input.
- The VM display-mode question explicitly explains `Headless` and `Visual`.
- Question JSON files now include `ExpectedAnswer` text for the user-facing prompts.
- `Oryn.sh help` now includes the same answer guide.

## Policy

Diagnostics and Panic remain mandatory and always enabled. User-selected modules still exclude the modules needed to get the kernel running.

- Generator options are displayed on an `OPTIONS` line and coloured in interactive terminals.
