## Build and run

From the Oryn repository root:

```bash
./Oryn.sh build Doos
./Oryn.sh run Doos
```

The generated manifest records mandatory kernel modules separately from user-selected modules. Runtime, Diagnostics, Panic, Cpu, and ManifestLoader are linked automatically. Diagnostics and Panic are always enabled. The generated kernel prints `Hello from Doos` during boot. The VM display mode is chosen during generation and can be Headless or Visual.
