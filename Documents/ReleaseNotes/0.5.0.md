# Oryn 0.5.0

## Stage 5 complete

Oryn 0.5.0 adds Stage 5 as the default development stage.

Stage 5 introduces the runtime contract layer on top of the Stage 4 approved-module boundary:

- added the Runtime SDK API and native module;
- added the Panic SDK API and native module;
- added Stage 5 bindings for `Runtime.Initialize`, `Runtime.MarkKernelReady`, and `Panic.Halt`;
- added `OSes/Stage5/Source/Kernel.cs`;
- updated `Runqemu.sh` so Stage 5 is the default stage;
- linked Runtime and Panic native modules into the freestanding kernel;
- added Stage 5 compiler and QEMU tests;
- rewrote `README.md` for Stage 5.

Stage 5 keeps user-facing code in the safe C# subset while unsafe halt, panic, runtime, CPU, memory, and diagnostics details remain inside approved Oryn modules.
