# Oryn 2.0.1 Release Notes

Oryn 2.0.1 fixes the first OrynVisualConfigurator polish issue after the 2.0.1 visual-first milestone.

## Fixed

- Removed the nullable warning in `Applications/OrynVisualConfigurator/Program.cs` caused by `Dictionary.TryGetValue` assigning to a non-nullable `string`.
- Replaced the terminal-only visual configurator with a dependency-free local browser UI.
- The configurator still reads the current version's `Questions/*.question.json` files and does not hardcode the question set.
- Choice questions render as browser drop-downs.
- Multi-choice questions render as browser check boxes.
- OS project search, load-directory, create-new, and configure-existing flows are available from the browser start screen.
- Added `ORYN_VISUALCFG_TERMINAL=1` / `--terminal` fallback for automation.

## Verification available in this package

- Shell syntax checks for `Oryn.sh`, `Runqemu.sh`, and `update.sh`.
- Packaging checks under `Tests/Generator/2.0.1/`.
