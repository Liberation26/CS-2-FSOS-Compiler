# Oryn 0.0.1 Release Notes

## Purpose

This release fixes the updater workflow for the starter Oryn filesystem archive.

## Changes

- Bumped the archive version to `0.0.1`.
- Updated `update.sh` so it extracts the downloaded `Oryn-*.zip` archive into a temporary directory first.
- The updater now copies `ChangedFiles/` from the temporary extraction directory into `~/Dev/OrynFoundry`.
- The updater no longer expects `ChangedFiles/` to already exist inside the source tree.
- The updater commits the copied files to the configured GitHub remote.

## Repository Target

The updater commits to:

```text
https://github.com/Liberation26/C--2-FSOS-Compiler.git
```

The default local working tree is:

```text
~/Dev/OrynFoundry
```
